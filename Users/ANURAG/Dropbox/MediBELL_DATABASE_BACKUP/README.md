# Privacy-Preserving IoT Healthcare System using Differential Privacy

## Overview
This project implements a disease prediction system for IoT-based healthcare that preserves patient privacy using Local Differential Privacy (LDP).

## Methodology
- IoT devices generate symptom indicators
- Local Differential Privacy is applied using randomized response
- A Random Forest classifier predicts diseases using privacy-preserved data
